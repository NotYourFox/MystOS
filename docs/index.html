О системе

MystOS - простая операцоинная система на ассемблере и Си

Создана больше для изучения работы операционной системы, чем для дела :)

На данный момент система не умеет ничего.

Скачивание и установка

MystOS можно запустить как на виртуальной машине, так и на реальной (не советуем).

Собранная операционная система находится в папке bin, а точнее - в файле mystos.bin. Скопируйте его содержимое на флешку или еще куда-то, если хотите запустить систему на реальной машине:

Внимательно проверьте аргументы при использовании dd! Не зря dd зовут Disk Destroyer! Все на Ваш страх и риск!
sudo dd if=./mystos.bin of=/флешка
Или используйте этот файл как жесткий диск для эмулятора: 
qemu-system-i386 -hda mystos.bin

Сборка

Вам не хватает собранных файлов в репозитории? Тогда соберите систему самостоятельно.

Для начала установите кросс-компилятор при помощи скрипта из репозитория.

Клонируйте репозиторий MystOS

git clone https://github.com/NotYourFox/MystOS

Отредактируйте файл build.sh


#/bin/bash
export PREFIX="$HOME/opt/cross" # путь к кросс-компилятору
export TARGET=i686-elf
export PATH="$PREFIX/bin:$PATH"
make all

Далее запустите build.sh, в папке bin будут свежие бинарники.
